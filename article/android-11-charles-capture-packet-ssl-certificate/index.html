使用 Charles 抓 Android 11 的包 ssl 证书问题

2021年08月24日 WindowsNetWork 约230字 预计需要1分钟

Android 7 以后，Android 不信任用户安装的证书，所以抓 https 时无法解码请求，对于第三方应用，需要将证书添加为系统证书，网上解决方法较多，比如使用安卓模拟器（兼容性差，很多应用闪退），使用 xposed 框架等，这里使用 Android 手机添加证书。

ps: 需要手机有 root 权限，以下操作以 Redmi 为准，其他品牌手机操作可能有差别。

导出证书（以 Charles 为例）

help --> SSL Proxying --> Save Charles Root Certificate…

Android 证书存储格式

证书路径

/system/etc/security/cacerts

文件命名格式（如果计算出来的 hash 值已经存在则编号依次 + 1）：<hash>.%d

将导出的证书计算 hash 值

需要安装 openssl

根据导出格式 2 选 1

.cer 格式证书

.pem 格式证书
